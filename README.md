# 历史模拟器：民国 (1912-1949)

🏛️ AI驱动的民国历史模拟游戏 - 改写历史，由你决定

## 🎮 游戏特色

- ✅ **真实历史事件驱动** - 基于1912-1949年真实历史事件
- 🎯 **多维度决策系统** - 军事、经济、外交、文化等7大维度
- 🔄 **动态历史演变** - 每个决策都会影响国家命运
- 🎨 **精美Web界面** - 现代化游戏UI，浏览器即可畅玩
- 📊 **实时国力可视化** - 动态进度条展示国力变化

## 📚 事件库

- 政治事件：50+
- 军事战役：12+  
- 外交事件：7+
- 经济事件：4+
- 社会变革：17+
- 文化事件：25+
- 科技进步：3+
- 人物事件：17+

**总计135+历史事件，持续扩充中...**

## 🎯 玩法说明

玩家扮演民国时期决策者，面对历史关键节点做出选择：

- 🎖️ **加强军事力量** - 提升军事实力，但消耗经济
- 💰 **发展经济建设** - 增加经济和民众支持
- 🤝 **改善外交关系** - 提升外交关系和威望
- 📚 **推动教育文化** - 发展教育文化和威望
- ⏭️ **维持现状** - 不做改变，观察局势

每个回合推进3个月，触发真实历史事件，根据决策改变国家发展轨迹！

## 🚀 快速开始

### 方式一：Web版游戏（推荐）

1. **安装依赖**
```bash
pip install flask flask-cors
```

2. **启动游戏服务器**
```bash
cd minguo-history-simulator
python3 web_server.py
```

3. **开始游戏**
在浏览器中打开：http://localhost:5001

就这么简单！在浏览器中享受精美的游戏界面 🎉

### 方式二：命令行版游戏

```bash
cd minguo-history-simulator
python3 game_demo.py
```

终端交互式游戏体验。

## 📊 国力系统

游戏包含7个维度的国力管理：

| 维度 | 说明 | 初始值 |
|------|------|--------|
| 🎖️ 军事实力 | 军队战斗力和防御能力 | 30 |
| 💰 经济 | 国家经济发展水平 | 25 |
| 🏛️ 威望 | 国际声望和影响力 | 40 |
| 👥 民众支持 | 人民对政府的支持度 | 50 |
| 🤝 外交关系 | 与列强的外交关系 | 35 |
| 📚 教育 | 教育普及和人才培养 | 20 |
| 🎭 文化 | 文化发展和思想进步 | 30 |

**失败条件**：
- 民众支持 ≤ 10：民心尽失，政权崩溃
- 经济 ≤ 5：经济崩溃，国家破产

**胜利条件**：
- 成功引导民国走过37年（1912-1949）

## 🏆 评级系统

根据最终综合国力评分（满分700）：

- **SSS** (600+): 完美统治！
- **SS** (500-599): 卓越领导！
- **S** (400-499): 出色表现！
- **A** (300-399): 良好发展
- **B** (200-299): 勉强及格
- **C** (<200): 亟需改进
